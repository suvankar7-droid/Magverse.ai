### Still seeing "Demo Mode"?

1. **Check file name:**
   ```
   Must be: .env
   Not: env.txt or .env.txt
   ```

2. **Check location:**
   ```
   Project root (not in src/ or public/)
   ```

3. **Check content:**
   ```
   VITE_OPENROUTER_API_KEY=sk-or-v1-...
   (No spaces around =)
   ```

4. **Restart server:**
   ```bash
   Ctrl + C
   npm run dev
   ```
